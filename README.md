calificacionesAlumnos
=====================

Aplicacion web segura en java que permite a alumnos y profesores la gestion de sus calificaciones. Se compone de una interfaz en la cual los profesores podran cargar notas de sus alumnos y visualizar las mismas. Los alumnos podran ver sus notas por asignatura y cuatrimestre.
